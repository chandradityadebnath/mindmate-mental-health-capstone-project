# System Architecture

## Overview
The Mental Health Agent System uses a multi-agent architecture with specialized AI agents working in parallel to provide comprehensive mental health support.

## Core Components

### 1. Agent System
- **Crisis Detection Agent**: Identifies emergency situations
- **Emotion Analysis Agent**: Analyzes emotional state
- **Support Planning Agent**: Creates therapeutic strategies
- **Resource Matching Agent**: Connects to external resources

### 2. Memory System
- Session management
- Long-term user memory
- Progress tracking

### 3. Safety System
- Content validation
- Crisis escalation
- Ethical guidelines

## Data Flow
1. User input received
2. Parallel agent processing
3. Response synthesis
4. Safety validation
5. Memory storage
6. Response delivery
